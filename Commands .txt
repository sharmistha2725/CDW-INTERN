GIT
----------------------------------------------------------------------------------------------------
1. git log : All previous logs
2. git pull : to pull repos (all changes from repo to local)
3. git fetch : brings files from external repo to local repo and use "git merge" to local directory
4. git add : to staging
5. git commit -m "msg" : to move to local repo
6. git push : to move to git repo
7. git checkout -b "branch name" : creates a new branch
8. git checkout branch name : shifts to a diff branch
9. git branch : lists all the branches
10. git status : status of the files
11. git merge "to which branch name(new)" : merge changes to new
